# Architecture Overview

This folder holds architecture notes for the LangGraph Company Intelligence Platform.

- `app/`: application entrypoint and runtime bootstrap.
- `agents/`: research and consolidation agent scaffolds.
- `validation/`: deterministic validation suite grouping.
- `persistence/`: Supabase persistence and repository abstractions.
- `pipelines/`: retry and orchestration scaffolds.
- `config/`: environment loading and configuration.
- `utils/`: shared helper modules.
